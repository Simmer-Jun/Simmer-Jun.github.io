    每一道题都传到jsfiddle上了，可以在结果项那里点击查看结果。
    在开始你的JavaScript8级考试前，整理好你的思绪，enjoy it!

题目

No.1

以下JavaScript表达式的结果是？

["1", "2", "3"].map(parseInt)

这是因为数组的map方法传的参数依次为item, index, array(数组的每一项， 每一项的索引， 原数组)，而parseInt方法则只接受2个参数string, radix前一个表示解析的字符串， 后一个表示解析的基数。

No.2

以下JavaScript表达式的结果是？

[typeof null, null instanceof Object]

对于typeof null来讲，始终会返回object，记住就好！

以下JavaScript表达式的结果是？
