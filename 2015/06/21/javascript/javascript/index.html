第四章 对象

对象基础

对象是JavaScript的基本数据类型。对象可以看作是属性的无序集合，每个属性都是一个名/值对。属性名是字符串，因此可以把对象看成是从字符串到值的映射。
除了名字和值之外，每个属性还有一些与之相关的特性，称为”熟悉特性”（property attribute）:

  • 可写（writable attribute），表明是否可以设置该属性的值。
  • 可枚举（enumerable attribute），表明是否可以通过for/in循环返回该属性
  • 可配置（configurable attribute），表明是否可以删除或修改该属性。

在ECMAScript5之后，对象还拥有三个相关的对象特性:

  • 对象的原型（prototype）指向另外一个对象，本对象的属性继承自它的原型对象。
  • 对象的类（class）是一个标识对象类型的字符串。
  • 对象的扩展标记（extensible flag）指明了（在ECMAScript5中）是否能向该对象添加新属性。

原型

所有通过对象字面量形式创建的对象都具有一个原型对象，可以通过Object.prototype获得对原型对象的引用。
通过关键字new和构造函数调用创建的对象的原型就是构造函数的prototype属性的值。所以有：

  1. new Array()创建的对象的原型就是Array.prototype，而Array.prototype的属性又继承于Object.prototype。这一系列的对象就是所谓的原型链（prototype chain）
  2. new Date()创建的对象的原型就是Date.prototype，而Date.prototype的属性又继承于Object.prototype。
  3. Object.prototype是为数不多的没有原型的对象之一。

有以下几种情况给对象o设置属性p会失败

  • o中的属性P是只读的：不能给只读的属性赋值（defineProperty()方法中有一个例外）。
  • o中的属性p是继承属性，并且是只读的：不能通过同名自有的属性覆盖只读的继承属性。
  • o中不存在自有属性p，O没有使用setter方法继承属性p，并且o的可扩展（extensible flag）性为false。

第五章 数组

对于JavaScript的数组，我们能想到的时什么？

是类型的引用，有各种特定的方法与属性，可以使用字面量和构造器来构造。

关于构造器与原型链还有闭包等见第6/7章（等待更新）

下面就数组的属性与方法进行讲解：

属性

  • constructor 制定创建对象的原型函数.
  • index 如果数组是通过正则表达式匹配创建的，匹配是字符串的下标索引为0.
  • input 如果数组是通过正则表达式匹配创建的，返回原始的字符串.
  • length 数组长度.
  • prototype 允许为所有对象附加属性.

Mutator方法（可以改变数组本身）

  • pop 移除数组的最后一个元素，返回值是被删除的元素。
  • push 在数组的末尾添加一个或者多个元素，返回值是新的数组的长度。
  • reverse 颠倒数组中元素的顺序，原先第一个元素现在变成最后一个，同样原先的最后一个元素变成了现在的第一个，也就是数组的索引发生了变化。
  • shift 删除数组的第一个元素，返回值是删除的元素。
  • unshift 添加一个或者多个元素在数组的开头，返回值是新的数组的长度。
  • splice 添加或删除数组中的一个或多个元素。

Accessor 方法 （不改变数组本身）

  • concat 返回一个包含此数组和其他数组和/或值的结合的新数组。
  • indexOf 返回第一个与给定参数相等的数组元素的索引，没有找到则返回-1。
  • join 将所有的数组元素连接成一个字符串。
  • lastIndexOf 返回在数组中搜索到的与给定参数相等的元素的最后（最大）索引。
  • slice 返回数组中的一段。
  • toString 返回代表该数组及其元素的字符,重写Object.toString 过程.

循环（迭代）过程

  • filter 对数组中的每一个元素调用参数中指定的过滤函数，并将对于过滤函数返回值为true的那些数组元素集合为新的数组返回。
  • forEach 对数组的每一个元素依次调用参数中指定的函数。
  • every 如果数组中每一个元素都满足参数中提供的测试函数，则返回真。
  • map Creates a new array with the results of calling a provided function on every element in this array.
  • some 如果数组中至少有一个元素满足参数函数的测试，则返回true。
