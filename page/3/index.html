JavaScript中的this（未完成待续）

我理解的this

JavaScript 中函数的调用有以下几种方式：作为对象方法调用，作为函数调用，作为构造函数调用，和使用 apply 或 call 调用。

在JavaScript中没有其他面对对象编程语言中的类的概念，而是以其特有的原型链方式来实现’类’的继承。而说到继承优不得不谈到JavaScript中的this关键字。

我个人理解的就是this指代的就是函数运行时函数所在的环境。这个和函数的闭包类似但有有些不同（恰恰相反），我们知道一个函数运行在它所定义的环境，而不是运行他的环境（详情可以看这里）。
