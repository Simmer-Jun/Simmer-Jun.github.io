CSS之玩转栅格布局

2016年9月5日
CSS

css布局

众所周知css作为层叠样式表能够决定页面的展现样式，一个页面的css按照实现的功能不同可以划分为以下几个部分:

  • 强制重置浏览器的默认样式：CSS Reset （也有类似normalize.css流的温和派）
  • 类似清除浮动等全局可以复用的样式（.clearfix ）还有自定义icon font字体图标等
  • 布局样式
  • 具体实现的内容样式

何为栅格布局？

引用Bootstrap关于栅格布局的原话:

Bootstrap includes a powerful mobile-first grid system for building layouts of all shapes and sizes. It’s based on a 12 column layout and has multiple tiers, one for each media query range. You can use it with Sass mixins or our predefined classes.

简单的概括就是：通过定义好的栅格类结合CSS3的媒体查询来让实现页面布局。
